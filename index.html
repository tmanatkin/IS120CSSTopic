CSS Measurement Elements

Pixels (px)

Pixels are an absolute length that are relative to the viewing device. For low-dpi devices, 1px is one device pixel (dot) of the display. For printers and high resolution screens 1px implies multiple device pixels.

Viewport Width (vw)

Relative to 1% of the width of the viewport. The viewport is the whole browser window.

See the Pen Untitled by jarommollinet (@jarommollinet) on CodePen.

Viewport Height (vh)

Relative to 1% of the height of the viewport. The viewport is the whole browser window.

See the Pen Untitled by jarommollinet (@jarommollinet) on CodePen.

Root Font Size (rem)

Relative to font-size of the root element, this can be set in your css. The default root font size is 16px.
